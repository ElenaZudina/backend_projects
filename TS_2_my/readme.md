# TS Kanban TODO

Простое веб-приложение на TypeScript для планирования задач в стиле Kanban (доска задач).
Позволяет создавать задачи с приоритетами, перемещать их между колонками и сохранять локально в браузере.

## Зависимости

Для работы и разработки проекта нужны:

- **Node.js** - установка пакетов и запуск сборки
- **TypeScript** - компилятор TypeScript

## Установка зависимостей

```bash
npm install
```

## Dev-зависимости

```json
{
    "devDependencies": {
        "typescript": "^5.6.3"
    }
}
```

## Сборка проекта

```bash
npm run build
```

## Запуск приложения

Приложение можно открыть через браузер. Например, если используете Live Server VS Code, оно будет доступно по адресу: [http://localhost:5500](http://localhost:5500)


## Функционал

- Создание задачи с выбором приоритета (Low, Medium, High)
- Колонки для задач: To do, Work in progress, Test, Done
- Перемещение задач между колонками с помощью Drag & Drop
- Подсветка завершенных задач: когда задача перемещается в Done, карточка кратковременно подсвечивается
- Удаление задач по кнопке на карточке
- Автоматическое сохранение задач в localStorage: все изменения сохраняются и восстанавливаются при перезагрузке
- Автоматическая дата создания и обновления задач
